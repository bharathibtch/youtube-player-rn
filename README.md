# YouTube Video Player

## Steps to Run

1. Install Expo Go app on your phone from App Store/Play Store

2. Install dependencies:
```bash
yarn install
```

3. Start Expo:
```bash
npx expo start
```

4. Scan QR code with your mobile Expo Go app

5. Done! App will run on your phone
